Security

Enterprise-Grade Protection

Most AI assistants run commands blindly. JARVIS has a full capability-based security framework that prevents destructive operations, scans installed skills for malware, and logs everything.

  • ✓ Capability-based tool permissions
  • ✓ Blocked paths (~/.ssh, ~/.aws, system dirs)
  • ✓ Blocked commands (rm -rf, sudo rm, format)
  • ✓ Skill sandboxing with malware scan
  • ✓ Full audit trail with log export
# JARVIS Security in action
You: Delete the .ssh folder

JARVIS: 🛡️ Operation blocked
Path ~/.ssh is in the protected list.
This directory contains SSH keys and
credentials. I can't modify it.

→ Logged to audit trail
Intelligence

Smart Cost Optimization

Why pay cloud prices for "hello world"? JARVIS automatically classifies task complexity and routes simple questions to free local models (Ollama), while sending genuinely complex reasoning to cloud providers.

  • ✓ Auto-classifies task complexity
  • ✓ Free local models for simple tasks
  • ✓ Cloud AI only when needed
  • ✓ Up to 90% cost reduction
  • ✓ Transparent routing decisions
# Cost-saving routing

Simple: "What's the Node version?"
→ Ollama (local, free) · 0.2s

Complex: "Refactor auth to use JWT"
→ Gemini Pro (cloud) · 3.1s

API costs this month: $1.40
vs $14.00 without tiered routing
Memory

It Actually Remembers

JARVIS has a 4-layer hierarchical memory system. It remembers your coding style, past conversations, lessons learned from mistakes, and builds up knowledge about your project over time.

  • ✓ Working memory (current session)
  • ✓ Episodic memory (past conversations)
  • ✓ Semantic memory (facts & knowledge)
  • ✓ Vector memory (similarity search)
  • ✓ Preference learning (adapts to you)
